# 🗄️ LeetCode SQL 50

A collection of SQL solutions to the [LeetCode SQL 50](https://leetcode.com/studyplan/top-sql-50/) study plan — a curated set of 50 SQL problems designed to strengthen query-writing skills from beginner to advanced level.

## 📖 About

This repository contains SQL solutions organized as individual `.sql` files. Each file includes:

- The original problem description and table schema (as a comment)
- An input/output example
- The SQL solution query

|---|---------|------------|------|

## 🚀 Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/ezequielfeliz/leetcode-sql-50.git
   cd leetcode-sql-50
   git checkout exercises
   ```

2. Open any `.sql` file to read the problem statement and solution.

3. Each file is self-contained — the schema and example data are documented inside as comments.

## 🛠️ Tech Stack

- **Language:** SQL (MySQL compatible)
- **Platform:** [LeetCode](https://leetcode.com)

## 📚 Study Plan Topics Covered

The LeetCode SQL 50 study plan covers the following categories:

- Select
- Basic Joins
- Basic Aggregate Functions
- Sorting and Grouping
- Advanced Select and Joins
- Subqueries
- Advanced String Functions / Regex / Clause
